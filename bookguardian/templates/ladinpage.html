O que é ?

O Book Guardian é um site open source com a finalidade de uma biblioteca virtual onde o usuário possa armazanear seus livros, mangás e HQs, editando-os de forma que lhes permite ver quais de seus livros já foram lidos, remove-los, ver suas descrições e muito mais!!

Esse projeto foi desenvolvido por alunos da faculdade UNIFG de Guanambi-BA, nele foi utilizado as linguagens: HTML, CSS, JavaScript, Python e seu framework Django.
