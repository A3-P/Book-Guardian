O que é ?

O Book Guardian é um site open source com a finalidade de uma blibioteca virtual onde o usuario possa armazanear seus livros, mangás e hq's, editando-os de forma que permitam-lhes ver quais de seus livros já foram lidos, remover livros, ver suas descrições e muito mais!!

Esse projeto foi desenvolvido por alunos da faculdade UNIFG de Guanambi-BA, nele foi utilizado as linguagens: HTML, CSS, JavaScript, Python e seu framework Django.
